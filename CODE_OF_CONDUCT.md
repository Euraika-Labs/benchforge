# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful, constructive, and specific.
- Assume good intent while still addressing impact.
- Accept feedback gracefully.
- Keep discussions focused on the project and its users.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without explicit permission.
- Spam, trolling, or repeated off-topic disruption.
- Any behavior that would make contributors feel unsafe or unwelcome.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code.

If you need to report a conduct concern, contact a repository maintainer privately through GitHub.
